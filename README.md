# CryptoAcademy

Landing page for a crypto and trading education platform. Dark tech aesthetic with cyan accents, Inter + Space Grotesk typography, course program, speaker profiles, pricing, and FAQ.

## Tech Stack

- **Next.js 16** — App Router, Turbopack
- **Tailwind CSS v4** — `@theme` directive for custom design tokens
- **Framer Motion** — Scroll-triggered animations
- **Lucide React** — Icon system

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
app/components/
  Header.tsx     — Fixed navbar with mobile hamburger menu
  Hero.tsx       — Hero section with course introduction
  Program.tsx    — Course curriculum breakdown
  Speakers.tsx   — Instructor profiles
  Pricing.tsx    — Course pricing plans
  Reviews.tsx    — Student testimonials
  FAQ.tsx        — Frequently asked questions accordion
  Footer.tsx     — Site footer with navigation
```
